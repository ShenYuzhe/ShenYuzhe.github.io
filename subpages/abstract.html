SIFT

Scale-invariant feature transform (SIFT) is an graphic alorithm from computer vision. It is designed to describe images by extracting and describing image feautures. It can be applied in many industrial fields including: picture matching, picture joining or even face detecting.


GPU

Graphic Processing Unit (GPU) is a special chip designed to manipulate and utilize built-in memory to accelerate the creation of images. Nowadays, because of its SIMD feature and the capacity of high parallelism, it is utilized to accelerate algorithms with huge computational complexity. APIs and frameworks are built to utilize this hardware. Users can easily use these APIs in their program.


Acceleration

Scale-invariant feature transform (SIFT) is a very complex algorithm involving huge amount of matrix or mathmetical computation which may take huge amount of time to process the result of images. So we utilize GPU to accelerate this algorithm.
